# AGENTS

Repo rules.

- `src/` is source of truth.
- `src/shared/` holds reusable fragments.
- `src/agents/_developer.toml` is main source agent.
- `build.sh` compiles into `opencode/agents/*.md` and `codex/agents/*.toml`.
- `install-opencode.sh` links OpenCode config and agents.
- `install-codex.sh` links Codex agents.
- `_backups/` is archival. Leave unchanged.
