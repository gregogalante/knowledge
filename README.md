# Knowledge

`src/` is source of truth. Build writes into `opencode/agents/*.md`, `codex/agents/*.toml`, and `codex/config.toml`.

## Build agents

```bash
chmod +x build.sh
./build.sh
```

## Setup for OpenCode

```bash
chmod +x install-opencode.sh
./install-opencode.sh
```

## Setup for Codex

```bash
chmod +x install-codex.sh
./install-codex.sh
```
